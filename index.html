    Installation

    Requirements

    jCoverflip depends upon: 1. jQuery version 1.3 or later 1. jQuery UI 1.7.2 (only the core of jQuery UI is required and the themes are not used)

    Installation

    1. Install jQuery 1.3+ and jQuery UI 1.7.2
    2. Download jCoverflip

    3. Add the styles to your site:

      .ui-jcoverflip {
  position: relative;
}

.ui-jcoverflip--item {
  position: absolute;
  display: block;
}

    4. Include jCoverflip script after jQuery and jQuery UI in the <head> of your HTML document:

      <script type="text/javascript" src="jquery.js"></script>
<script type="text/javascript" src="jquery.ui.js"></script>
<script type="text/javascript" src="jquery.jcoverflip.js"></script>
